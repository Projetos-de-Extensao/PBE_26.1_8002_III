    Gestão de Documentação de Estágio¶

    Automatize a validação de estágios obrigatórios e a gestão de documentos da Secretaria do Ibmec com IA e uma API RESTful moderna.

    Acessar Repositório Site da Disciplina

    O Problema¶

    • Processo manual, repetitivo e sujeito a erros
    • Dificuldade em lidar com muitas versões de documentos
    • Excesso de requerimentos abertos
    • Comunicação entre agentes dos processos muito manual

    Nossa Solução¶

    • Automatizar tarefas repetitivas e pouco variáveis
    • Banco de dados para armazenar os documentos
    • API RESTful para gestão de arquivos e validação de horas
    • IA Generativa para analisar contratos e gerar insights
    • Comunicação centralizada entre agentes

    Tecnologias Utilizadas¶

    Python

    Linguagem base da aplicação

    Django & DRF

    Core do backend e API REST

    MySQL

    Banco de Dados relacional

    IA Generativa

    LLaMA & Gemini API para análise de documentos
